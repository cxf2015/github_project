# Paper Logger - 论文阅读记录系统

一个基于 Web 的本地论文阅读和笔记管理工具，帮助您高效地管理和记录学术论文阅读过程。

## ✨ 功能特性

- 📖 **PDF 阅读** - 支持打开本地 PDF 文件和在线 URL PDF
- 📝 **实时笔记** - 右侧笔记区域，随时记录阅读心得
- 💾 **持久化存储** - 所有论文和笔记自动保存到 SQLite 数据库
- 🔄 **历史记录** - 下次打开自动显示之前的论文和笔记
- 🎯 **页面关联** - 笔记可以关联到具体的 PDF 页码
- ✏️ **编辑管理** - 支持编辑和删除笔记
- 📊 **统计展示** - 显示每篇论文的笔记数量
- 🚀 **轻量级** - 基于 Flask + SQLite，无需复杂配置

## 🛠️ 技术栈

- **后端**: Python Flask
- **数据库**: SQLite
- **前端**: HTML5 + CSS3 + JavaScript
- **PDF 渲染**: 浏览器内置 PDF 查看器

## 📦 安装步骤

### 1. 环境要求

- Python 3.7+
- 现代浏览器 (Chrome, Firefox, Edge 等)

### 2. 安装依赖

```bash
cd paper_logger
pip install -r requirements.txt
```

### 3. 运行应用

```bash
python app.py
```

应用将在 `http://localhost:5000` 启动。

### 4. 访问应用

在浏览器中打开 `http://localhost:5000` 即可使用。

## 📖 使用指南

### 添加论文

#### 方式一：通过侧边栏添加
1. 点击左侧边栏的 "**+ 添加论文**" 按钮
2. 输入论文标题
3. 选择文件来源类型：
   - **本地文件**: 选择 PDF 文件上传
   - **URL 链接**: 输入 PDF 的在线地址
4. 点击 "创建论文记录"

#### 方式二：快速添加本地 PDF
1. 点击顶部工具栏的 "**📁 打开本地 PDF**" 按钮
2. 选择 PDF 文件
3. 输入论文标题
4. 自动创建记录并打开

#### 方式三：快速添加 URL PDF
1. 点击顶部工具栏的 "**🔗 打开 URL PDF**" 按钮
2. 输入 PDF 的 URL 地址
3. 输入论文标题
4. 自动创建记录并打开

### 记笔记

1. 在左侧选择一篇论文
2. 右侧会自动显示该论文的所有笔记
3. 在底部文本框中输入笔记内容
4. 可选：填写页码以关联到具体页面
5. 点击 "**💾 保存笔记**" 按钮

### 管理笔记

- **编辑笔记**: 点击笔记右上角的 "✏️ 编辑" 按钮
- **删除笔记**: 点击笔记右上角的 "🗑️ 删除" 按钮
- **查看历史**: 切换不同论文时，笔记会自动加载对应内容

### 数据持久化

所有数据存储在 `papers.db` SQLite 数据库文件中，包括:
- 论文标题
- 文件路径/URL
- 所有笔记内容及页码
- 创建和更新时间

即使关闭应用，下次打开时所有数据都会自动恢复。

## 📁 项目结构

```
paper_logger/
├── app.py                 # Flask 后端主程序
├── requirements.txt       # Python 依赖列表
├── papers.db             # SQLite 数据库 (运行时生成)
├── uploads/              # 上传的 PDF 文件存储目录
├── templates/
│   └── index.html        # 前端页面模板
└── README.md            # 本说明文档
```

## 🔧 配置选项

在 `app.py` 中可以修改以下配置:

```python
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///papers.db'  # 数据库路径
app.config['UPLOAD_FOLDER'] = 'uploads'  # 上传文件夹
app.config['MAX_CONTENT_LENGTH'] = 50 * 1024 * 1024  # 最大上传文件大小 (50MB)
app.run(debug=True, port=5000)  # 运行端口和调试模式
```
