# Paper Logger 翻译功能配置指南

## 🌐 翻译功能说明

中英文对照翻译功能允许您在阅读英文论文时，将选定的文本翻译成中文，实现左右分屏对照阅读。

**新增特性**: 支持 4 种主流翻译引擎，可根据需求灵活选择！

## 📋 支持的翻译引擎

| 引擎 | 特点 | 免费额度 | 推荐场景 | 配置难度 |
|------|------|----------|----------|----------|
| **百度翻译** | 稳定可靠，学术文献翻译准确 | 200 万字符/月 | ⭐⭐⭐⭐⭐ 学术论文 | ⭐⭐ |
| **有道翻译** | 中英文翻译准确，支持多语言 | 50 万字符/月 | ⭐⭐⭐⭐ 日常使用 | ⭐⭐⭐ |
| **Google 翻译** | 全球最广泛使用 | 完全免费 | ⭐⭐⭐ 快速翻译 | ⭐ (无需配置) |
| **DeepL** | 准确度最高，专业级 | 付费服务 | ⭐⭐⭐⭐⭐ 高质量需求 | ⭐⭐ |

## 🔧 配置步骤

### 方式一：使用 Google 翻译 (最简单 - 推荐新手)

**无需任何配置，开箱即用！**

1. 直接在翻译界面选择 "Google 翻译"
2. 输入文本即可翻译
3. ⚠️ 注意：国内可能需要网络代理才能访问

### 方式二：配置百度翻译 (推荐 - 稳定可靠)

#### 1. 申请百度翻译 API 密钥

1. 访问百度翻译开放平台：**https://fanyi-api.baidu.com/**
2. 注册/登录百度账号
3. 进入"管理控制台"
4. 点击"开通服务"
5. 选择"通用翻译 API"
6. 完成实名认证 (个人开发者即可)
7. 创建应用获取 APP ID 和密钥

#### 2. 配置 API 密钥

打开 `app.py` 文件，找到以下配置项:

```python
# 翻译 API 配置
# 百度翻译 API 注册地址：https://fanyi-api.baidu.com/
app.config['TRANSLATION_APP_ID'] = ''  # 填入你的百度翻译 APP ID
app.config['TRANSLATION_SECRET_KEY'] = ''  # 填入你的百度翻译密钥
```

填入你申请到的 APP ID 和密钥:

```python
app.config['TRANSLATION_APP_ID'] = '20240101000000001'  # 示例 APP ID
app.config['TRANSLATION_SECRET_KEY'] = 'YourSecretKeyHere'  # 示例密钥
```

### 方式三：配置有道翻译

#### 1. 申请有道翻译 API 密钥

1. 访问有道智云：**https://ai.youdao.com/**
2. 注册/登录网易账号
3. 进入"控制台"
4. 创建应用，选择"自然语言翻译服务"
5. 获取 APP ID 和密钥

#### 2. 配置 API 密钥

在 `app.py` 中配置:

```python
# 有道翻译 API 注册地址：https://ai.youdao.com/
app.config['YOUDAO_APP_ID'] = ''  # 填入你的有道翻译 APP ID
app.config['YOUDAO_SECRET_KEY'] = ''  # 填入你的有道翻译密钥
```

### 方式四：配置 DeepL 翻译 (专业级 - 付费)

#### 1. 购买 DeepL API

1. 访问 DeepL API：**https://www.deepl.com/pro-api**
2. 注册 DeepL 账号
3. 选择订阅计划 (推荐 DeepL API Pro)
4. 获取 API Key

#### 2. 配置 API 密钥

在 `app.py` 中配置:

```python
# DeepL 翻译 API 注册地址：https://www.deepl.com/pro-api
app.config['DEEPL_API_KEY'] = ''  # 填入你的 DeepL API 密钥 (付费)
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 重启应用

停止当前运行的应用，重新启动:

```bash
python app.py
```

或双击 `start.bat`

## 🎯 使用方法

### 选择翻译引擎

1. 打开翻译面板 (点击"🈯️ 翻译"按钮)
2. 在底部找到"🤖 选择翻译引擎"下拉框
3. 选择你想要的翻译引擎
4. 系统会自动显示该引擎的特点说明

### 开始翻译

1. 在输入框中输入英文内容
2. 点击"🈯️ 立即翻译"
3. 翻译结果显示在右侧面板
4. 原文和译文对照显示，清晰易读

## 💡 免费额度说明

### 百度翻译
- **标准版**: 每月免费 200 万字符
- **QPS**: 10 次/秒
- 对于个人学习和研究用途完全够用

### 有道翻译
- **免费版**: 每月 50 万字符
- **QPS**: 10 次/秒
- 适合中低频使用

### Google 翻译
- **完全免费**: 无限制
- **限制**: 有频率限制，不适合批量翻译
- 适合偶尔使用

### DeepL
- **免费试用**: 50 万字符/月 (需注册)
- **付费版**: €9.99/月起
- 质量最高，适合专业需求

## ⚠️ 注意事项

1. **API 调用频率**: 避免短时间内大量调用 API，可能被限制
2. **文本长度**: 单次翻译建议不超过 2000 字符
3. **网络要求**: 
   - 百度/有道：国内可正常访问
   - Google：需要网络代理
   - DeepL：国内可直接访问
4. **密钥安全**: 不要将包含密钥的代码上传到公开代码仓库
5. **依赖库**: 使用有道翻译需要安装 pycryptodome

## 🔧 故障排除

### 问题 1: 提示"翻译 API 未配置"

**解决方案**: 
- 检查 `app.py` 中对应的 API 密钥是否已填写
- 确保没有多余的空格或引号
- 重启应用

### 问题 2: 提示"签名错误"

**解决方案**:
- 检查密钥是否正确复制
- 确认密钥没有包含多余的空格
- 在对应平台控制台重新生成密钥

### 问题 3: 提示"QPS 超限"

**解决方案**:
- 降低翻译频率
- 等待几秒后再试
- 考虑升级到更高 QPS 的版本

### 问题 4: Google 翻译无法使用

**解决方案**:
- 检查网络连接
- 可能需要配置网络代理
- 或者切换到百度/有道翻译

### 问题 5: 有道翻译报错

**解决方案**:
- 确保已安装 pycryptodome: `pip install pycryptodome`
- 检查有道 API 版本是否为 v3
- 查看有道控制台的错误码说明
